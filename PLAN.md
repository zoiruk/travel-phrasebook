# Проект: Туристический разговорник (RU→EN) — веб-приложение

Дата старта: 2026-07-04 · Отдельный проект (НЕ часть «English for Seasonal Workers»).

## Что и зачем

Бумажный «Русско-английский разговорник» (Victory / «Виктория плюс», СПб, 2004, серия
«Галопом по Европам», ISBN 5-7062-0180-3) физически рассыпается. Владелец хочет перенести
его в электронный формат как офлайн веб-приложение.

**Метод — ГИБРИД (решение владельца 2026-07-04):** структуру и охват тем берём из книги,
но английский пишем заново (современный British English), транскрипцию — свою по
конвенциям. Книга = карта тем, НЕ источник текста.

## 🔴 Копирайт — почему именно гибрид

Копирайт-страница книги: «Исключительное право публикации принадлежит ООО "Виктория плюс".
Выпуск без разрешения — противоправен». → **Дословно копировать книгу нельзя** (её
составительский текст защищён). Темы/ситуации и отдельные слова копирайтом не защищены,
поэтому:
- Русские заголовки-темы и «что нужно уметь сказать» — берём как карту.
- Каждую английскую фразу и её транскрипцию — АВТОРИМ заново, современно.
- Тогда приложение — самостоятельное произведение, можно выкладывать публично.

Устаревшее из книги НЕ тащим: `How do you do`, `It's a great honour to make your
acquaintance` → современно: `Hello`, `Nice to meet you`.

## Стек (переиспользуем проверенный подход фермерского проекта)

Vanilla HTML/CSS/JS, офлайн-first, без фреймворков/сервера/БД. Озвучка — Web Speech API
(`lang='en-GB'`, cancel-before-speak). Файлы: `index.html` (инлайн CSS+JS) + `data.js`
(`window.PHRASEBOOK`). Открывается по `file://`, деплой — GitHub Pages (как фермерский).

Схема данных:
```js
window.PHRASEBOOK = [{
  id, icon, title_ru,
  phrases: [{ ru, en, transcr }]   // transcr — кириллица по конвенциям
}]
```

## Полная карта разделов (из содержания книги, с.236–239 + начало)

Начало (до с.78) — общие фразы, приветствия/знакомство, погранконтроль/таможня, транспорт
(самолёт/поезд). Далее по содержанию:

1. **Общие фразы · Приветствия · Знакомство** (👋)
2. **Погранконтроль. Таможня** (🛂)
3. **В городе:** поиск адреса · автобус · метро · такси · надписи (🗺️)
4. **Проживание в гостинице:** оформление · пребывание · просьбы к горничной · жалобы ·
   отъезд (🏨)
5. **Обмен валюты** (💱)
6. **Ресторан:** столовые принадлежности · приправы · способы приготовления · пицца ·
   напитки (🍽️)
7. **Покупки:** продавец · ткани · книжный · электротовары · одежда · парфюм/галантерея ·
   фото · универсам · продукты (🛍️)
8. **Развлечения. Отдых:** кино · театр · ночной клуб · музей · ТВ · за город · спорт (🎭)
9. **Почта. Телеграф** (📮)
10. **Телефон** (📞)
11. **Сервис:** парикмахерская · обувная мастерская · ремонт часов · прачечная/химчистка ·
    фотомастерская (✂️)
12. **Медицинская помощь:** терапевт · хирург · стоматолог · аптека · оптика (🏥)
13. **Проблемы** (⚠️)
14. **О себе:** имя/возраст · местожительство · семья · работа · хобби (🙋)
15. **Образование** (🎓)
16. **Экономическое сотрудничество · Выставки/ярмарки** (💼) — *низкий приоритет, деловое*
17. **Приложение 1 — меры:** длина/площадь/объём/вес · время/даты · дни · месяцы · сезоны ·
    погода · деньги (📏)
18. **Приложение 2:** цвета · числительные (колич./порядк./дроби) · проценты (🔢)

Некоторые разделы книги устарели/нерелевантны (фотомастерская, телеграф, «экономическое
сотрудничество») — на Фазе приоритизации решим: включать как есть, осовременить (телеграф→
интернет/почта) или пропустить.

## Заметки по качеству

- Английский — естественный современный BrE, не пословный перевод русского. Идиоматичность
  важнее буквальности («Сколько стоит?» → `How much is it?`, не `How much does it cost?`
  везде).
- Транскрипция — кириллица по конвенциям (как в фермерском проекте): `/w/`→«у», `θ`→«с»,
  `ð`→«з», конечная `r` в BrE не читается. Сомнительные — свериться, не выдумывать.
- Объём книги ~3000 фраз — это МНОГО. Не гнать всё сразу; ценность даёт качество разделов,
  которые владелец реально откроет в поездке (транспорт, гостиница, деньги, еда, медицина,
  проблемы), а не «технические выставки».
